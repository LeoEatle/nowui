# 目录规范

```
component
|---assets //其他资源
|     |---index.scss
|---example //使用示例
|      |---.npmignore
|---lib //生成的发布文件
|    |---index.js
|---src //源文件
|    |---index.js
|---test  //测试文档目录
|    |---setup.js
|    |---test.js
|---package.json
|---README.md
|---.babelrc  //babel 配置
|---.npmignore  // 发布 npm 时的忽略项
``` 

test/setup.js 是用于测试环境模拟浏览器对象使用。
