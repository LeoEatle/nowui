# now-base-empty 空白页

支持 5 种 empty 样式, 通过传入的 emptyType 控制, 默认垂直居中

## API

| 属性          | 说明                                 | 类型     | 默认值 | 是否必传 |
| ------------- | ------------------------------------ | -------- | ------ | -------- |  
| emptyType |  'default' 'search', 'chat', 'gift', 'error' 中选择一个   | 可选  |  'default'      | 否       |


## 安装

```
npm install now-base-empty
```

## 传参示例

``` 
import Empty from 'now-base-empty'

<Empty>
  显示的文字
</Empty>
```
